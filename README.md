# tdex-analytics
Analytics and Historical data feed about TDEX Network

### Usage 
Build binaries:
```
make build
```
Provision:
```
make provision
```
Config CLI:
```
./bin/tdexa config
```
List balances:
```
./bin/tdexa balances --from_time "2022-02-08T14:34:40+01:00"
```
List prices:
```
./bin/tdexa prices --from_time "2022-02-08T14:34:40+01:00"
```
